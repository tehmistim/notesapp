# Notes app

Create public and private notes using an interactive app with modern styling and easy to use functionality.

This app uses 
- Node.js
- Express
- MongoDB
- Google OAuth authentication

    ✔️ Login using google account
    ✔️ Create public/private notes
    ✔️ Delete/Edit notes
